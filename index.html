üèóÔ∏èMotherboard (Papan Induk)

Motherboard adalah papan sirkuit utama yang menghubungkan semua komponen komputer. Seperti "tulang punggung" komputer yang menyediakan jalur komunikasi antar komponen.

Komponen Utama:

  • ‚Ä¢ Socket CPU (tempat processor)
  • ‚Ä¢ Slot RAM (tempat memory)
  • ‚Ä¢ Slot PCIe (untuk VGA, WiFi card)
  • ‚Ä¢ Konektor SATA (untuk HDD/SSD)
  • ‚Ä¢ Port I/O (USB, Audio, LAN)

Tips Memilih:

  • ‚úìSesuaikan socket dengan CPU yang dipilih
  • ‚úìPerhatikan jumlah slot RAM
  • ‚úìPastikan ada slot PCIe untuk VGA
  • ‚úìCek ketersediaan port yang dibutuhkan

üß†CPU (Central Processing Unit)

CPU adalah "otak" komputer yang menjalankan semua instruksi dan perhitungan. Kecepatan CPU diukur dalam GHz (Gigahertz).

Spesifikasi Penting:

  • ‚Ä¢ Core: Jumlah inti pemrosesan (2, 4, 6, 8+)
  • ‚Ä¢ Thread: Jalur pemrosesan virtual
  • ‚Ä¢ Clock Speed: Kecepatan dalam GHz
  • ‚Ä¢ Cache: Memory sementara CPU
  • ‚Ä¢ Socket: Jenis konektor (LGA, AM4)

Brand Populer:

Core i3, i5, i7, i9

AMD

Ryzen 3, 5, 7, 9

üíæRAM (Random Access Memory)

RAM adalah memory sementara yang menyimpan data yang sedang digunakan. Semakin besar RAM, semakin banyak program yang bisa berjalan bersamaan.

Jenis RAM:

  • ‚Ä¢ DDR4: Standard saat ini (2133-3200 MHz)
  • ‚Ä¢ DDR5: Generasi terbaru (4800+ MHz)
  • ‚Ä¢ Kapasitas: 4GB, 8GB, 16GB, 32GB+

Rekomendasi Kapasitas:

Office/Browsing 8GB
Gaming 16GB
Content Creation 32GB+

üéÆGPU/VGA (Graphics Processing Unit)

GPU bertanggung jawab untuk memproses grafik dan visual. Ada yang terintegrasi di CPU (integrated) dan ada yang terpisah (dedicated).

Jenis GPU:

Built-in di CPU, cukup untuk office & browsing

Dedicated GPU

Kartu grafis terpisah, untuk gaming & rendering

Brand Populer:

AMD

Tips: Untuk gaming 1080p, pilih GPU mid-range. Untuk 4K atau content creation, pilih GPU high-end.

‚ö°PSU (Power Supply Unit)

PSU mengubah listrik AC dari rumah menjadi DC yang dibutuhkan komponen komputer. Kualitas PSU sangat penting untuk stabilitas sistem.

Rating Efisiensi:

80+ Bronze 82-85% efisiensi
80+ Gold 87-90% efisiensi
80+ Platinum 90-92% efisiensi

Kalkulasi Wattage:

Gaming PC 550-750W
High-end PC 850W+

Tips: Pilih PSU dengan wattage 20-30% lebih tinggi dari kebutuhan untuk efisiensi optimal.

‚öôÔ∏èPanduan Instalasi Komputer

‚ö†Ô∏èPersiapan Sebelum Instalasi

  • ‚Ä¢Siapkan tools: obeng Phillips, anti-static wrist strap (opsional)
  • ‚Ä¢Baca manual motherboard dan komponen lainnya
  • ‚Ä¢Pastikan area kerja bersih dan terang
  • ‚Ä¢Lepaskan listrik statis dengan menyentuh logam
1

Persiapan Case & Motherboard

Persiapan Case:

  • ‚Ä¢ Buka kedua sisi panel case
  • ‚Ä¢ Lepas standoff yang tidak perlu
  • ‚Ä¢ Pasang I/O shield motherboard ke case
  • ‚Ä¢ Siapkan kabel power dan data

Persiapan Motherboard:

  • ‚Ä¢ Keluarkan motherboard dari box
  • ‚Ä¢ Letakkan di atas anti-static bag
  • ‚Ä¢ Periksa socket dan slot
  • ‚Ä¢ Baca diagram pin header
2

Instalasi CPU & Cooler

Memasang CPU:

  1. Buka socket CPU dengan mengangkat retention arm
  2. Keluarkan CPU dari kemasan (jangan sentuh pin/contact)
  3. Sejajarkan tanda segitiga pada CPU dengan socket
  4. Letakkan CPU dengan lembut (jangan ditekan)
  5. Turunkan retention arm untuk mengunci

Memasang Cooler:

  1. Oleskan thermal paste tipis dan merata (jika perlu)
  2. Pasang mounting bracket sesuai socket
  3. Letakkan heatsink di atas CPU
  4. Kencangkan sekrup secara diagonal dan bertahap
  5. Hubungkan kabel fan ke header CPU_FAN
3

Instalasi RAM

  1. Buka clip pengunci slot RAM
  2. Periksa notch (lekukan) pada RAM dan slot
  3. Untuk dual channel, gunakan slot 2 & 4 (biasanya berwarna sama)
  4. Masukkan RAM dengan sudut 90¬∞ hingga clip mengunci otomatis
  5. Pastikan RAM terpasang rata dan kencang

Tips: RAM membutuhkan sedikit tenaga untuk masuk. Jika sulit, periksa orientasi dan jangan dipaksa.

4

Instalasi Storage (SSD/HDD)

M.2 SSD:

  1. Cari slot M.2 di motherboard
  2. Lepas sekrup mounting
  3. Masukkan SSD dengan sudut 30¬∞
  4. Tekan ke bawah dan kencangkan sekrup

SATA SSD/HDD:

  1. Pasang drive ke bracket case
  2. Kencangkan dengan sekrup
  3. Hubungkan kabel SATA data ke motherboard
  4. Hubungkan kabel SATA power dari PSU
5

Instalasi GPU (Jika Ada)

  1. Lepas slot cover PCIe di case yang sesuai
  2. Buka clip pengunci slot PCIe x16 (slot teratas)
  3. Masukkan GPU hingga terdengar klik
  4. Kencangkan bracket GPU ke case dengan sekrup
  5. Hubungkan kabel power PCIe dari PSU (jika diperlukan)

Penting: GPU high-end membutuhkan kabel power 6-pin atau 8-pin dari PSU.

6

Instalasi PSU & Kabel

Memasang PSU:

  1. Tentukan orientasi fan PSU (ke bawah jika case ada filter)
  2. Masukkan PSU ke compartment
  3. Kencangkan dengan 4 sekrup dari belakang case

Koneksi Kabel Utama:

  • ‚Ä¢ 24-pin ATX: Power utama motherboard
  • ‚Ä¢ 8-pin CPU: Power CPU (kiri atas motherboard)
  • ‚Ä¢ PCIe 6/8-pin: Power GPU (jika ada)
  • ‚Ä¢ SATA Power: Power untuk SSD/HDD
7

Koneksi Front Panel & Final Check

Front Panel Connectors:

‚Ä¢ Power Button
‚Ä¢ Reset Button
‚Ä¢ Power LED
‚Ä¢ HDD LED
‚Ä¢ USB Headers
‚Ä¢ Audio Headers

*Lihat diagram di manual motherboard untuk posisi yang tepat

Final Check:

  • ‚úì Semua kabel power terpasang
  • ‚úì RAM terkunci dengan benar
  • ‚úì GPU terpasang kencang
  • ‚úì Tidak ada sekrup yang tertinggal
  • ‚úì Cable management rapi
8

First Boot & BIOS Setup

Langkah First Boot:

  1. Hubungkan monitor ke GPU (bukan motherboard)
  2. Hubungkan keyboard dan mouse
  3. Colokkan kabel power dan nyalakan PSU
  4. Tekan tombol power case
  5. Masuk BIOS dengan menekan Del/F2 saat booting

Setting BIOS Penting:

  • ‚Ä¢ Enable XMP/DOCP untuk RAM
  • ‚Ä¢ Set boot priority (USB untuk install OS)
  • ‚Ä¢ Check temperature CPU & fan speed
  • ‚Ä¢ Enable UEFI mode
  • ‚Ä¢ Save & Exit

üîçTroubleshooting & Tips

‚ùåKomputer Tidak Menyala

Kemungkinan Penyebab:

  • ‚Ä¢ PSU tidak nyala atau rusak
  • ‚Ä¢ Kabel power tidak terpasang
  • ‚Ä¢ Front panel connector salah
  • ‚Ä¢ RAM tidak terpasang dengan benar
  • ‚Ä¢ Short circuit di motherboard

Solusi:

  • ‚Ä¢ Cek switch PSU (posisi ON)
  • ‚Ä¢ Pastikan kabel 24-pin & 8-pin CPU terpasang
  • ‚Ä¢ Periksa koneksi power button
  • ‚Ä¢ Lepas dan pasang ulang RAM
  • ‚Ä¢ Test dengan 1 stick RAM saja

‚ö†Ô∏èNyala Tapi Tidak Ada Display

Kemungkinan Penyebab:

  • ‚Ä¢ Monitor terhubung ke motherboard bukan GPU
  • ‚Ä¢ RAM tidak terdeteksi
  • ‚Ä¢ GPU tidak terpasang dengan benar
  • ‚Ä¢ CPU atau motherboard bermasalah

Solusi:

  • ‚Ä¢ Hubungkan monitor ke port GPU
  • ‚Ä¢ Reseat RAM (lepas dan pasang ulang)
  • ‚Ä¢ Coba 1 stick RAM di slot berbeda
  • ‚Ä¢ Pastikan GPU terkunci di slot PCIe
  • ‚Ä¢ Clear CMOS jika perlu

üå°Ô∏èOverheating Issues

Gejala:

  • ‚Ä¢ Komputer restart sendiri
  • ‚Ä¢ Performance menurun drastis
  • ‚Ä¢ Fan berputar sangat kencang
  • ‚Ä¢ Blue screen atau freeze

Solusi:

  • ‚Ä¢ Bersihkan debu dari heatsink dan fan
  • ‚Ä¢ Ganti thermal paste CPU
  • ‚Ä¢ Tambah case fan untuk airflow
  • ‚Ä¢ Periksa mounting cooler CPU
  • ‚Ä¢ Monitor temperature dengan software

üí°Tips Maintenance

Maintenance Rutin:

  • ‚Ä¢ Bersihkan debu setiap 3-6 bulan
  • ‚Ä¢ Monitor temperature secara berkala
  • ‚Ä¢ Update driver dan BIOS
  • ‚Ä¢ Backup data penting
  • ‚Ä¢ Check kabel dan koneksi

Tools yang Berguna:

  • ‚Ä¢ HWiNFO64 (monitoring hardware)
